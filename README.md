# Site Currículo (HTML/CSS/JS) — PT/EN

## Como abrir
1) Extraia o ZIP
2) Dê duplo clique em `index.html`

✅ Não precisa Node, npm, nada.

## Onde editar
- Links/telefone/email: `app.js` (objeto `profile`)
- Textos PT/EN: `app.js` (objeto `i18n`)
- Projetos do portfólio: `app.js` (array `projects`)
- Estilos: `styles.css`
- Imagens dos projetos: `assets/projects/`

## Dica (opcional)
Se quiser abrir como um “servidor” (pra evitar bloqueios de arquivo local),
use a extensão do VSCode: **Live Server** e clique em “Go Live”.
